#  Alura Store — Análisis de Ventas

Challenge 1 de Data Science | ONE (Oracle Next Education) + Alura Latam

##  Descripción

Análisis de rendimiento de las 4 tiendas del Sr. Juan para identificar cuál tiene el menor desempeño y recomendarla como candidata a venta, permitiéndole invertir en un nuevo negocio.

##  Análisis realizados

1. **Facturación total** — ¿Cuál tienda genera más ingresos?
2. **Ventas por categoría** — ¿Qué productos se venden más en cada tienda?
3. **Calificación promedio** — ¿Cómo evalúan los clientes cada tienda?
4. **Productos más y menos vendidos** — ¿Qué funciona y qué no?
5. **Costo de envío promedio** — ¿Cuánto cobra cada tienda por el envío?

##  Cómo ejecutar

1. Abre el archivo `AluraStoreLatam.ipynb` en [Google Colab](https://colab.research.google.com)
2. Ejecuta todas las celdas con `Ctrl + F9`
3. Los datos se cargan automáticamente desde el repositorio de Alura

##  Tecnologías usadas

- Python 3
- Pandas
- Matplotlib
